# Agente IA n8n + Playwright — Color Express

Sistema de cotización automática con IA para imprentas que usan Smartier.

## Stack
- n8n (self-hosted) — orquestador del flujo
- Claude (Anthropic) — IA conversacional
- Playwright — automatización web de Smartier
- Telegram Bot API — canal de comunicación
- Node.js + Express — servidor HTTP

## Archivos principales
- `cotizar.js` — automatización Playwright
- `servidor.js` — servidor HTTP Express (puerto 3001)
- `docs/` — documentación completa del proyecto

## Documentación
Ver carpeta `/docs` para arquitectura, workflow n8n, system prompt y más.
